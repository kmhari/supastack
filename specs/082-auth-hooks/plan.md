# Implementation Plan: Auth Hooks (hook_*) — pg-functions:// Phase 1

**Branch**: `082-auth-hooks` | **Date**: 2026-05-25 | **Spec**: [spec.md](spec.md)

**Input**: Feature specification from `specs/082-auth-hooks/spec.md`

## Summary

Promote all 21 `hook_*` auth-config fields from `stored_only` to `honored` in the env-field-mapper, activate the already-commented-out `GOTRUE_HOOK_*` env vars in the compose template, add cross-field validation to reject non-`pg-functions://` URI schemes and enabled-without-URI combos, and add a dashboard **Auth → Hooks** page with per-hook toggle + URI + secrets form.

No new microservices. No database migrations. GoTrue's native `pg-functions://` dispatch handles the runtime side.

### B. Backend: runtime-config-store.ts — cross-field validation

**File**: `apps/api/src/services/runtime-config-store.ts`

In `crossFieldValidate()`, after the existing OAuth provider check, add:

```typescript
// Hook URI scheme + enabled-requires-URI validation (feature 082 / issue #64)
const HOOK_TYPES = [
  'custom_access_token', 'mfa_verification_attempt', 'password_verification_attempt',
  'send_sms', 'send_email', 'before_user_created', 'after_user_created',
] as const;

for (const hookType of HOOK_TYPES) {
  const uriField = `hook_${hookType}_uri` as string;
  const enabledField = `hook_${hookType}_enabled` as string;
  const uri = merged[uriField];
  const enabled = merged[enabledField];

  if (uri != null && uri !== '' && !String(uri).startsWith('pg-functions://')) {
    const isHttps = String(uri).startsWith('https://') || String(uri).startsWith('http://');
    throw new ManagementApiError(
      400,
      isHttps
        ? 'HTTPS hook URIs are not yet supported. See issue #64 for progress.'
        : 'Hook URI scheme not supported. Only pg-functions:// is accepted (Phase 1). HTTPS support tracked in issue #64.',
      'hook_uri_scheme_unsupported',
      { field: uriField },
    );
  }

  if (enabled === true && (uri == null || uri === '')) {
    throw new ManagementApiError(
      400,
      'A URI is required when a hook is enabled.',
      'hook_uri_required',
      { field: uriField },
    );
  }
}
```
