## Design Decisions (from research.md)

| Decision | Rationale |
|---|---|
| Test `classifyInstance` indirectly via exported functions | Private function; SC-004 requires zero source changes |
| `runSingleInstanceReconcile` for `forceRetry` + `instance_gone` (deleting) | Only exported path that passes `forceRetry=true` to `classifyInstance` |
| `runFullReconcile` for `orphan_in_supavisor` | Only code path that produces this classification |
| `startRun` for preflight + concurrency | `preflight()` is private and called at the top of `startRun` |
| `vi.useFakeTimers()` for `failed_stale` boundary | `classifyInstance` uses `Date.now()` directly |
| `vi.mock` builder chain returning configurable arrays | Matches T030/T031 proven pattern |

## Test Structure

### Describe blocks

```
pooler-reconciler
├── classifyInstance (via runFullReconcile + runSingleInstanceReconcile)
│   ├── consistent — active pooler row + sv tenant present
│   ├── missing_pooler_row — instance with no pooler row
│   ├── pg_password_drift — pooler row.status === 'pg_password_drift'
│   ├── missing_in_supavisor — active pooler row, no sv tenant
│   ├── orphan_in_supavisor — sv tenant with no matching instance
│   ├── instance_gone (deleting) — inst.status === 'deleting' via runSingleInstanceReconcile
│   ├── instance_gone (not found) — no row in supabase_instances
│   └── failed_stale boundary (fake timers)
│       ├── age > 1h → failed_stale
│       ├── age === 1h → consistent (strict > not >=)
│       └── forceRetry bypasses staleness → failed_stale
├── runFullReconcile — remediation isolation
│   ├── per-instance failure → partial_failure, others proceed
│   ├── auth-class error → maybePromoteToDrift → pg_password_drift
│   └── actions_taken aggregates correctly
└── startRun — preflight + concurrency
    ├── stale running row → flipped to failed with worker_crash_detected
    ├── GC sweep → executes DELETE (assert db().execute called)
    └── concurrent insert → ReconcilerInFlightError with id + startedAt
```
